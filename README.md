# Engenharia-Social
Um repositório criado pra demonstração do que foi aprendido na Dio, no tópico de engenharia social 
aqui eu falei sobre uma das ferramentas do kali linux que permite a criação de uma pagina falsa pra conseguir a senha ou credencial do usuario.

começamos com comando Sudo su para utilização do super administrador do kali linux 
Depois de digitar a senha do seu user,digite setoolkit e iria abrir um menu 
escolhemos a primeira opção chamada de  Social-Engineering Attacks
após escolhermos irá abrir outro menu escolhemos a segunda opção chamada de Website Attack Vectors
vamos de Credential Harvester Attack Method e depois Site Cloner havendo tambem a opçao de criar um template original pra engenharia social
com isso precisamos confirma em qual ip o servidor ira habituar ,confirma com enter
depois selecionamos o site do qual iremos clonar 
eu selecionei o instagram e por final coloquei no programa depois só coloquei meu ip no site e o site já havia sido clonado 
ainda estou aprendendo a analisar as informações no terminal 

Já deixo avisado que o que foi ensinado aqui é totalmente e exclusivamente pra uso didatico.
